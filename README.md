# 🌊 Amethyst Tide: Interactive WebGL Water Ripples

A beautiful, interactive landing page experience featuring fluid water distortions and procedurally animated SVG fish.

## ✨ Features
- **Interactive Water Physics:** A custom WebGL fragment shader via Three.js that responds dynamically to mouse movements and clicks.
- **Glassmorphism Aesthetic:** Modern, sleek UI design with deep purple and lavender gradients.
- **Fluid Blob Loader:** A pure CSS loading sequence featuring morphing, spinning gradients.
- **Procedural SVG Fish:** Dynamic DOM-based SVG fish that swim continuously, adapting their colors to the current theme while perfectly refracting underneath the water shader.

## 🛠️ Tech Stack
- **HTML5:** Layout structure and advanced SVG `<feTurbulence>` filters.
- **CSS3:** Custom animations, layout, and styling.
- **JavaScript (ES6):** Logic and state management.
- **Three.js (WebGL):** 3D context and custom shader materials.

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/MDAwaiz438/Water-Ripple-Effect.git
   ```
2. Serve the directory locally. You can use any local server, such as:
   ```bash
   npx serve .
   ```
3. Open the provided `localhost` URL in your browser to experience the ripples.
